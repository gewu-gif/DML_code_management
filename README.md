#This is a GPU-accelerated file for Interpretable Double Machine Learning.

This file covers the entire workflow of Interpretable Double Machine Learning, including Shapley analysis, Pearson correlation analysis, control variable selection, model comparison, model computation, and IPW robustness test.

Before running the code, you need to set up an accelerated environment based on RAPIDS and PyTorch.
